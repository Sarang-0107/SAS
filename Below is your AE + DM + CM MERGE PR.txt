🔹 AE count per subject
SELECT USUBJID, COUNT(*) AS AE_COUNT
FROM AE
GROUP BY USUBJID;

🔹 Any Serious AE
SELECT USUBJID,
       CASE WHEN SUM(CASE WHEN AESER = 'Y' THEN 1 ELSE 0 END) > 0
            THEN 'Y' ELSE 'N' END AS ANY_SERIOUS_AE
FROM AE
GROUP BY USUBJID;

🔹 Rescue Medication Flag
SELECT USUBJID,
       CASE WHEN SUM(CASE 
            WHEN UPPER(MEDNAME) IN ('IBUPROFEN','DICLOFENAC') 
            THEN 1 ELSE 0 END) > 0
       THEN 'Y' ELSE 'N' END AS USED_RESCUE_MED
FROM CM
GROUP BY USUBJID;

🔹 Dose Change Flag
SELECT USUBJID,
       CASE WHEN COUNT(DISTINCT EXDOSE) > 1
            THEN 'Y' ELSE 'N' END AS DOSE_CHANGED
FROM EX
GROUP BY USUBJID;

🧠 WHAT IS ADaM? (Very Simple)

👉 ADaM = Analysis Data Model

SDTM → raw, event-level

ADaM → analysis-ready, subject-level

💡 One subject = one row (mostly)
💡 Used by statisticians & FDA reviewers

🔄 SDTM vs ADaM (Must Know)
SDTM	ADaM
Many rows per subject	One row per subject
AE, CM, EX events	Summary variables
Collection focused	Analysis focused
