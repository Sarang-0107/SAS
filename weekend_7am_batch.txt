SDTM Guidelines:

1.  As per SDTM Guidelines domain name should contain 2 letters

2. Every domain should have proper label -(40char) - prop case
	and in domain for each variable have proper label

3.  Variable names should not contains more than 8 char
		USUBJID
		AGE
		LBORRES
		LBORRESU

4. order should be as per SDTM IG 3.2

5. All SDTM date variables are char datatypes

	date value:  YYYY-DD-MM
			2021-01-08
	
	IS8601da.   : 2021-01-08
	IS8601dt.   : 2021-01-08T10:30:20
------------------------------------------------------------------------------
core:

1. Required(REQ): Variables must be included in domain.
			These values cant be null


		eg :  SUBJID  SEX RACE AGE
		        101   M   ASAIN 30
		        102	F     ASAIN  25



2. Expected(Exp) : variables must be included in domain.
			values can be missing or not missing

		eg :RFSTDTC
		    RFENDTC
		    RFXSTDTC
		    RFXENDTC


3. Permissible(perm) : values can be null or not null

	it depends on a sponsor to include or drop this variabel in a domain
	
		eg:    INVNAM
			INVID
