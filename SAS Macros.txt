1️⃣ Missing Value Check Macro (VERY COMMON)
🔹 What it does

Checks missing values for any dataset & variable.

🔹 Macro
%macro chk_missing(ds=, var=);

proc sql noprint;
  select count(*) into :miss_cnt
  from &ds
  where &var is missing;
quit;

%put NOTE: Missing count for &ds..&var = &miss_cnt;

%mend;

2️⃣ Study Day Calculation Macro ⭐ (VERY STRONG)
🔹 What it does

Calculates study day for any date variable.

🔹 Macro
%macro calc_studyday(eventdt=, refdt=, outvar=);

if &eventdt >= &refdt then
  &outvar = &eventdt - &refdt + 1;
else if &eventdt < &refdt then
  &outvar = &eventdt - &refdt;

%mend;

🔹 Explanation

“This macro ensures consistent study day calculation across domains like AE, VS, and CM.”
