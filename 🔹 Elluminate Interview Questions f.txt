🔹 What is MedDRA (MEDCDR)?

MedDRA = Medical Dictionary for Regulatory Activities

It is a standard medical dictionary used to code:

Adverse Events (AE)

Medical History (MH)

Indications (IND)

Concomitant Medications (sometimes)

Why MedDRA is used

✔ Consistent terminology
✔ Regulatory compliant (FDA, EMA)
✔ Easier analysis and reporting

🔹 What is MEDCDR in Elluminate?

MEDCDR = MedDRA Coder

In Oracle Health Sciences Elluminate, MEDCDR is the coding component used to:

👉 Code verbatim medical terms entered in eCRFs
👉 Map them to standard MedDRA terms

🔹 Example (Very Important)
Verbatim entered by site:
"Severe headache with nausea"

MEDCDR codes it to:
MedDRA Level	Term
LLT	Headache
PT	Headache
HLT	Headaches
HLGT	Nervous system disorders
SOC	Nervous system disorders
🔹 What is MEDCDR Merge in Elluminate?
🔑 Simple definition

MEDCDR Merge is the process of merging coded MedDRA data from MEDCDR back into the Elluminate clinical database.

👉 Coding happens in MEDCDR
👉 Data collection happens in Elluminate
👉 Merge = sync coding results back

🔹 Why is MEDCDR Merge needed?

Because:

Sites enter verbatim terms in Elluminate

Coders code them in MEDCDR

Elluminate must receive:

LLT

PT

SOC

MedDRA version

So MEDCDR Merge updates the clinical database with the coded values.

🔄 MEDCDR Merge – Step-by-Step Flow (Interview GOLD)
1️⃣ Data entry in Elluminate

Site enters:

AE Term = "Stomach pain"

2️⃣ Data sent to MEDCDR

Verbatim term sent for coding

Status = Not coded

3️⃣ Coding in MEDCDR

Coder assigns:

LLT = Abdominal pain
PT  = Abdominal pain
SOC = Gastrointestinal disorders

4️⃣ MEDCDR Merge back to Elluminate

Coded values populated in:

AELLT

AEDECOD

AESOC

Coding status updated

🔹 What gets merged during MEDCDR Merge?

Typically:

Coded term (LLT)

Preferred Term (PT)

SOC

MedDRA version

Coding status

Dictionary ID

After MEDCDR Merge
AE Verbatim	AEDECOD	AESOC

🔹 Common Interview Questions & Answers
Q1. What is MEDCDR?

A: MEDCDR is the MedDRA coding module used within Elluminate to code medical terms.

Q2. What is MEDCDR Merge?

A: MEDCDR Merge is the process of integrating coded MedDRA data back into the Elluminate database.

Q3. Why is MEDCDR Merge important?

A: It ensures coded terms are available in the clinical database for analysis and regulatory submission.

Q4. Which domains use MEDCDR?

A: AE, MH, IND (sometimes CM depending on study).

Q5. What happens if merge is not done?

A: Elluminate will still show verbatim terms but coded fields remain blank.
